# toggle-svelte

Toggle some content.

## slots

- `unnamed` Content that should be toggled

## props

- `summary - String | Default: ''`
  - Summary for toggled content.

## example

```html
<toggle-svelte summary="Svelte pros">
  <ul>
    <li>Nice</li>
    <li>Cool</li>
    <li>Soft</li>
  </ul>
</toggle-svelte>
```
